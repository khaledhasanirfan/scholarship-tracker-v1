# Scholarship Tracker

Trying to build a useful app for aspirants of higher studies who are looking for scholarships. This is the first noob version. Using Codex here fully

## Local development

Install dependencies:

```bash
npm install
```

Start the local website:

```bash
npm run dev
```

Build the production files:

```bash
npm run build
```

## Free hosting plan

This project is set up for GitHub Pages. When changes are merged into `main`, the GitHub Actions workflow in `.github/workflows/deploy.yml` builds the site and deploys the `dist` folder.

The live URL will be:

```text
https://khaledhasanirfan.github.io/scholarship-tracker/
```

If the first deployment does not start automatically, open the repository on GitHub, go to **Settings > Pages**, and set **Build and deployment > Source** to **GitHub Actions**.
